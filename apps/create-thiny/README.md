# create-thiny

> Scaffold a new Thiny AI agent project with one command

[![npm](https://img.shields.io/npm/v/create-thiny)](https://www.npmjs.com/package/create-thiny)

## Usage

```bash
pnpm create thiny my-agent
cd my-agent
pnpm install
pnpm dev
```

This scaffolds a complete Thiny project with:

- TypeScript configuration
- Pre-configured agent with safety middleware
- Environment variable template
- Ready-to-run examples
